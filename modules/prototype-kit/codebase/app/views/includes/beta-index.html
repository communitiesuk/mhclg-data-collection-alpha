Beta 1

Hypotheses we’re testing with this prototype

  1. If we show more than one thing per page in the household characteristics section:
    • we should observe users all completing the form as easily as the previous version of the prototype
    • users who utilise assive technology will be able to successfully navigate through the section
  2. By changing the "Exit to logs" link to "Exit to task list", we will observe users building a successful mental modal of the service IA

Release notes

  • Rebuilt the prototype to allow for easier question reordering and flow-logic.
  • All data is now stored in the session. You should use the clear data in the footer to reset the prototype.
  • This version of the prototype only contains questions in the “about the household” section.
  • Changed the javascript reveal of more questions (on same page) to separate pages in the household situation journey.
  • Removed the "Answer missing questions" component from the check answers pages which didn't test well last round.
  • Remove ALL validation until we can decided on how best to skip past questions the user might not know in the moment.
